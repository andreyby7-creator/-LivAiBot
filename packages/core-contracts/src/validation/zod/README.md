# @livai/core-contracts/src/validation/zod

## Validation / Zod

- **Backend** валидирует входные данные через **Pydantic**.
- **Zod** используется как **frontend/runtime guard** (формы/клиент), а не как замена backend-валидации.
- `generated/**` — **автогенерируется из OpenAPI** и **не редактируется вручную** (правки только через OpenAPI + генератор).
